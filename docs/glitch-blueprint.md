## Glitch Constitution

- Collectors come first.
- Simple beats complicated.
- Fast beats flashy.
- Beautiful matters.
- Automation should save time, not create confusion.
- Every feature must make collecting easier or more enjoyable.
- Protect the MVP from feature creep.

## Mission

Build the operating system for collectors: identify, organize, value, buy, sell, trade, discover, compete, and connect.

## Vision

Glitch will become the operating system for the collectibles industry.

Five-year goal: build the most trusted platform for collectors worldwide.

## Development Philosophy

Build in this order:

1. Build Now (MVP)
2. Build Next
3. Vision Vault

Do not build every idea immediately. Preserve all ideas in the backlog. Prioritize stability, speed, UX, scalability, then expansion.

## Build Now

1. AI photo identification
2. Inventory management
3. Search
4. Dashboard
5. User accounts
6. Market values
7. Mobile-first UX
8. Foundation for marketplace integrations

## Design Language

Modern premium aesthetic, mobile-first, white backgrounds with purple accents, smooth subtle animations, clean UX.

## Integration Targets

- eBay
- Whatnot
- TCGplayer
- Shopify
- BrickLink

## AI Vision

Natural language assistant that can answer inventory, market, event, and collecting questions and proactively surface opportunities.
